# CRUD Python + PostgreSQL

Projeto acadêmico de CRUD completo com interface gráfica em Python.

## Tecnologias utilizadas
- Python 3.14
- PostgreSQL 18
- psycopg2
- Tkinter
- Faker

## Funcionalidades
- Cadastrar produtos
- Listar produtos
- Atualizar produtos
- Excluir produtos

## Como rodar

1. Instale as dependências:
pip install psycopg2-binary faker

2. Configure o banco de dados no arquivo `appBD.py` com suas credenciais

3. Rode o `criar_tabela.py` para criar a tabela no banco

4. Rode o `appGUI.py` para abrir a interface gráfica
